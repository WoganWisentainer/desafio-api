API

A API Rest foi feita usando-se Python 3 (versão 3.14) usando FastAPI. Também foram usadas as bibliotecas Uvicorn e Requests. Ademais, o ChatGPT foi utilizado para fazer a atividade. 

No caso ela faz uma consulta sobre o link disponibilizado no desafio e vai retornar os dados juntamente da data usada. A data usada deve ser no formato Ano-mês-dia.

Passos:
1 – Baixar o arquivo do GitHub.
2 – Abrir o terminal CMD.  Abrir onde a pasta estiver. Para instalar as três bibliotecas (FastAPI, Uvicorn e Requests), deve ser utilizado o comando “pip install -r requirements.txt” no terminal. 
3 – Após a instalação das bibliotecas, executar com o comando “uvicorn main:app –reload”. Se tudo der certo, a API vai estar em “http://127.0.0.1:8000” e a documentação vai estar em “http://127.0.0.1:8000/docs”.
